Скрипт открывает bcf и создает html отчет с описанием замечаний, картинки, guid.

Как запустить через vs code:

Создайте виртуальное окружение: python -m venv venv
Активируй его: venv\Scripts\Activate
Установи зависимости: pip install -r requirements.txt
Запусти: python app.py

Откройте браузер: http://127.0.0.1:5000
